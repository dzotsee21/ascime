### ASCIME is an image-to-ascii art converter (now supports GIFs too)

commands:
<br />
`ascime <img_path> -s` - Use the SIMPLE_CHARS set (default is NORMAL_CHARS)
<br />
`asci <img_path> -w 200` - Set output width (default: 100)
<br />
`ascime <img_path> -c` - Generate colored ASCII output
<br />
`ascime <img_path> -c only` - Colored output using only the first character from the selected ASCII set
<br />
`asci <img_path> -c gray` - Generate grayscale-colored ASCII output
<br />

You can also combine flags:
<br />
`ascime <img_path> -c only -s -w 200`
